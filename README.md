## Instruções para executar o projeto

### Documentação em Postman dos endpoints
https://#

### Usando docker (opção 1)
A imagem do docker gera um container com o projeto

Start project:
```
docker-compose up
```

Stop project:
```
docker-compose down
```

### Executando localmente (opção 2)
```
npm install
npm run start
```
ou se preferir o yarn
```
yarn install
yarn start
```
